Study Types

Quick Study - You are given random questions, once you get one right you won't be asked it again. It ends when you get every question right once.

Long Study - You will do multiple rounds, in every round each question in your list is asked in a random order. It ends when you get every question in a round correct.

Test - You are asked each question once, regardless of whether you get it right or wrong.

Settings Explanation

Autocorrect - The user can choose what language they want autocorrect in or if they want it turned off.

Case Sensitive - By default the program ignores whether the user input is upper or lower case, turning this on makes the program check this.

Don't Correct - By default the program will tell the user what the correct answer was if they got it wrong. If this setting is turned on the program won't say what the correct answer was.

Re-Do - If you turn this on, when you get a question wrong the program will make you rewrite the correct answer.
